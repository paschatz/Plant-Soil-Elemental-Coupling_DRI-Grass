# Plant-Soil-Elemental-Coupling_DRI-Grass
This repository contains the R scripts, and output figures for the study on plant-soil elemental coupling in the DRI-Grass experiment.

## Project Structure
The repository is organized as follows:

- ```data/```: Contains the raw and processed datasets used for analysis.

```DRIGrass_coupling_data.csv:``` The exported dataset for coupling analysis.

```DRIGrass_tidy.data.csv:``` Cleaned/formatted version of the data.

- ```scripts/```: R scripts for data processing and statistical analysis.

```DRIGrass_data.cleaning.R```: Pre-processing steps to prepare raw data for analysis.

```DRIGrass_Coupling.analysis.R```: Main statistical analysis and modeling.

```DRIGrass_Sup_Mterial.R```: Analysis specifically for supplementary tables and figures.

- ```exports/```: Contains the final outputs generated by the scripts.

```fig1.tiff``` through ```fig6.tiff```: Primary publication figures.

```DRIGrass_Fig.S1.tiff```, etc.: Supplementary figures.

```plant_element_anova_results.csv```: Statistical output tables.

```DryGrass_ELM.Rproj```: The RStudio project file. Opening this ensures all file paths are relative and correct.

## Getting Started
### Prerequisites

You will need R installed on your system. We recommend using RStudio to open the ```.Rproj``` file.

How to Reproduce Analysis

- Clone the repository

- Run Cleaning Script: Execute ```scripts/DRIGrass_data.cleaning.R``` to ensure the environment is set up.

- Run Analysis: Execute ```scripts/DRIGrass_Coupling.analysis.R``` to generate the primary figures and tables found in the exports/ folder.
